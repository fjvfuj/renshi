# 人事管理系统 (HR Management System)

现代化企业OA系统，基于 Next.js + Node.js + MongoDB 构建。

## 🌐 在线预览

- **前端**: http://localhost:3001
- **后端API**: http://localhost:3000/api

## 🚀 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Next.js 14, TypeScript, Tailwind CSS |
| 后端 | Node.js, Express, MongoDB |
| 部署 | Docker, Docker Compose |

## 📱 功能模块

### 核心人事
| 模块 | 说明 |
|------|------|
| 👨‍💼 员工管理 | 员工档案增删改查 |
| 📋 考勤管理 | 每日签到签退 |
| 📝 请假管理 | 请假申请与审批 |
| 💰 薪酬管理 | 工资发放记录 |
| 🏢 部门管理 | 组织架构管理 |

### 行政办公
| 模块 | 说明 |
|------|------|
| 📢 通知公告 | 公司通知发布 |
| 🔄 审批流程 | 工作流审批 |
| 📅 会议管理 | 会议室预约 |
| 📁 文档管理 | 企业文档共享 |

### 人力资源
| 模块 | 说明 |
|------|------|
| ✅ 任务管理 | 任务分配与跟踪 |
| 📊 绩效考核 | 员工绩效评估 |
| 👥 招聘管理 | 候选人流程管理 |
| 📚 培训管理 | 培训课程安排 |
| 📦 资产管理 | 企业资产登记 |

### 系统
| 模块 | 说明 |
|------|------|
| ⚙️ 系统设置 | 个人与系统配置 |

## 🛠️ 本地开发

```bash
# 启动后端
cd backend && npm start

# 启动前端
cd frontend && npm run dev
```

## 🐳 Docker 部署

```bash
docker-compose up -d
```
