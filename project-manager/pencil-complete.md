# Pencil 原型设计工具 - 完整学习手册

## 一、Pencil 简介

Pencil 是一个**免费开源的 GUI 原型设计工具**，由 Evolus 公司开发。

### 主要特点
- ✅ 完全免费开源
- ✅ 支持 Windows、Linux、Mac
- ✅ 内置丰富的 UI 组件库
- ✅ 支持流程图绘制
- ✅ 可导出 PNG、SVG、PDF
- ✅ 轻量级，安装简单

---

## 二、安装指南

### Windows 安装
1. 访问 https://pencil.evolus.vn/Downloads.html
2. 下载 Windows 版本 (.exe)
3. 双击运行安装

### Mac 安装
1. 访问 https://pencil.evolus.vn/Downloads.html
2. 下载 Mac 版本 (.dmg)
3. 拖拽到 Applications 文件夹

### Linux 安装
```bash
# 方法1: Flatpak (推荐)
flatpak install flathub com.evolus.Pencil
flatpak run com.evolus.Pencil

# 方法2: DEB包
wget https://github.com/evolus/pencil/releases/download/v3.1.1/pencil_3.1.1_amd64.deb
sudo dpkg -i pencil_3.1.1_amd64.deb
sudo apt-get install -f -y

# 方法3: AppImage
wget https://github.com/evolus/pencil/releases/download/v3.1.1/Pencil-3.1.1.AppImage
chmod +x Pencil-3.1.1.AppImage
./Pencil-3.1.1.AppImage
```

## 六、设计流程

### 步骤1: 新建项目
1. 打开 Pencil
2. 选择 "File" → "New"
3. 选择空白页或模板

### 步骤2: 绘制页面结构
1. 从工具栏选择矩形
2. 在画布上绘制页面框架
3. 添加标题栏、导航栏

### 步骤3: 添加UI组件
1. 从组件库拖拽组件到画布
2. 调整位置和大小
3. 设置样式颜色

### 步骤4: 添加交互
1. 选中元素
2. 右键选择 "Action"
3. 设置跳转页面或动作

### 步骤5: 导出
1. "File" → "Export"
2. 选择格式 (PNG/SVG/PDF)
3. 保存文件

## 九、常见问题

### Q: Pencil 支持中文吗？
A: 支持，在文本工具中直接输入中文即可。

### Q: 如何添加自己的图片？
A: 插入 → 图片 → 选择本地文件

### Q: 可以导入其他工具的设计吗？
A: 可以导入 PNG、SVG 图片作为参考。

### Q: 如何制作交互原型？
A: 选中元素 → 右键 → Action → 设置跳转到某页面
