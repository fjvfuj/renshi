# Pencil 原型设计工具学习笔记

## 什么是 Pencil?

Pencil 是一个**免费开源的 GUI 原型设计工具**，可用于：
- 绘制流程图
- 创建线框图 (Wireframe)
- 设计 GUI 原型
- 制作交互式原型

## 下载安装

### 桌面版本
- **Windows**: 下载 .exe 安装包
- **Linux**: 下载 .deb 或 .rpm 包
- **macOS**: 下载 .dmg 安装包

## 基本功能

### 1. 界面组成
- **工具栏**: 选择绘图工具
- **画布**: 绘制原型区域
- **属性面板**: 设置元素属性
- **页面面板**: 管理多个页面

### 2. 常用工具
| 工具 | 用途 |
|------|------|
| 选择工具 | 选择和移动元素 |
| 文本工具 | 添加文字 |
| 形状工具 | 绘制矩形、圆形等 |
| 线条工具 | 绘制直线、箭头 |
| 图片工具 | 插入图片 |
| 连接器 | 连接元素 |

### 3. 模板库
- **Blank Page**: 空白页
- **Desktop**: 桌面应用模板
- **Mobile**: 手机应用模板
- **iOS**: iOS 设备模板
- **Android**: Android 设备模板

## 使用步骤

### 创建新项目
1. 打开 Pencil
2. 选择 "File" → "New" 
3. 选择模板或空白页

### 绘制界面
1. 从工具栏选择形状
2. 在画布上拖拽绘制
3. 调整大小和位置
4. 在属性面板设置样式

### 添加交互
1. 选择元素
2. 右键 "Action" 
3. 设置点击行为
4. 链接到其他页面

### 导出
- **File** → **Export**
- 支持 PNG, SVG, PDF 格式

## 项目中使用

Pencil 可用于：
1. **需求分析** - 绘制业务流程图
2. **原型设计** - 创建 UI 线框图
3. **团队沟通** - 导出给开发人员参考

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建 |
| Ctrl+S | 保存 |
| Ctrl+Z | 撤销 |
| Delete | 删除 |
| Ctrl+鼠标滚轮 | 缩放 |
