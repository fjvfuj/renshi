# Pencil 原型设计工具 - 深入学习

## 📥 下载安装

### Linux 安装
```bash
# 方法1: 使用deb包
wget https://github.com/evolus/pencil/releases/download/v3.1.1/pencil_3.1.1_amd64.deb
sudo dpkg -i pencil_3.1.1_amd64.deb
sudo apt-get install -f -y

# 方法2: 使用Flatpak
flatpak install flathub com.evolus.Pencil
```

### Windows/Mac
- 访问 https://pencil.evolus.vn/Downloads.html 下载对应版本

## 🎯 核心功能详解

### 1. 页面管理
- **新建页面**: Ctrl+N
- **复制页面**: 右键 → Duplicate
- **重命名**: 双击页面名称
- **页面切换**: 点击页面面板中的页面

### 2. 形状工具 (Shape Tools)

#### 常用形状分类
| 类别 | 形状 | 用途 |
|------|------|------|
| Basic | 矩形、圆形、线条 | 基础UI元素 |
| Text | 文本框 | 文字说明 |
| Flowchart | 流程图元素 | 业务流程 |
| UI Patterns | 按钮、输入框、表格 | 界面原型 |

#### 形状操作
- **移动**: 拖拽
- **调整大小**: 拖拽边缘手柄
- **旋转**: 拖拽顶部旋转手柄
- **对齐**: 选中多个 → 对齐工具

### 3. 文本编辑
- 双击文本框进入编辑模式
- 支持富文本格式
- 字体、大小、颜色可调

### 4. 样式设置
- **填充颜色**: 形状内部颜色
- **边框颜色**: 形状边缘颜色
- **边框宽度**: 线条粗细
- **透明度**: Alpha通道调整

## ⌨️ 快捷键速查

| 快捷键 | 功能 |

| Ctrl+Shift+Z | 重做 |
